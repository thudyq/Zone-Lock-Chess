# 一、游戏规则

## 基本规则

在一个 N × N 的棋盘中，两名玩家轮流落子。

- 黑棋先手

- 棋子一旦落下不可移动

- 同一格只能落子一次

---

## 合法落子条件（规则A）

某个空格对于玩家 P 合法，当且仅当：

```text
该格上下左右四个方向
不存在对方棋子
```

即：

- 可以贴着自己的棋子下

- 不可以贴着对方棋子下

例如：

```text
⚫ .
⚪ .
```

空格同时邻接黑白棋：

```text
⚫ □ ⚪
```

则：

```text
双方都不能下
```

成为永久废点。

---

## 游戏结束

当轮到某玩家时：

```text
若该玩家无合法落点
↓
立即结束游戏
```

然后判断：

### 情况1

对方仍有合法落点：

```text
对方获胜
```

### 情况2

对方也没有合法落点：

```text
平局
```
